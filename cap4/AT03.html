Glossário Web

HTML
Linguagem usada para estruturar páginas.
CSS
Linguagem usada para estilizar páginas.
JavaScript
Linguagem responsável pela interatividade.
API
Interface que permite comunicação entre sistemas.
Servidor
Computador que hospeda aplicações e sites.
Programa responsável por responder requisições.
DNS
Sistema que converte nomes em endereços IP.
Facilita a localização de sites.
Framework
Biblioteca
Ferramentas reutilizáveis para acelerar o desenvolvimento.
URL
Endereço Web
Identifica e localiza recursos na internet.
